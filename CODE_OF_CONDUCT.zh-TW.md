# 貢獻者公約行為準則 (Contributor Covenant Code of Conduct)

## 我們的承諾

我們作為成員、貢獻者和領導者，承諾讓參與我們社群的每個人都能享受一個無騷擾的體驗，無論其年齡、體型、可見或不可見的殘疾、種族、性別特徵、性別認同與表達、經驗水平、教育背景、社經地位、國籍、個人外貌、族群、階級、膚色、宗教，或是性取向。

我們承諾以貢獻於一個開放、歡迎、多元、包容且健康的社群為最高指導原則進行交流與互動。

## 我們的標準

有助於為我們社群創造積極環境的行為範例包括：

* 對他人展現同理心與善意
* 尊重不同的意見、觀點與經驗
* 提出並優雅地接受具建設性的反饋
* 承擔責任並向受到我們錯誤影響的人道歉，並從經驗中學習
* 專注於什麼對整個社群最為有利，而不僅是為了個人利益

不可接受的行為範例包括：

* 使用性化的語言或圖像，以及任何形式的性暗示或性關注
* 惡意挑釁、侮辱或貶損性評論，以及個人或政治層面的攻擊
* 公開或私下的騷擾
* 在未獲明確許可的情況下發布他人的私人資訊（例如實體地址或電子郵件地址）
* 其他在專業環境中可被合理視為不恰當的行為

## 執行責任

社群領導者有責任釐清並落實我們對可接受行為的標準，且將針對任何被其認定為不恰當、具威脅性、具冒犯性或有害的行為採取適當且公平的糾正行動。

社群領導者有權利及責任移除、編輯或拒絕與本行為準則不符的留言、提交 (commits)、程式碼、Wiki 編輯、Issues 以及其他貢獻，並將在適當時機溝通其審核決定的理由。

## 適用範圍

本行為準則適用於所有社群空間，同時也適用於當個人在公共空間中正式代表社群時的行為。
代表我們社群的範例包括：使用官方電子郵件地址、透過官方社群媒體帳號發文，或在線上/線下活動中擔任指定代表。

## 執行細則

如發生任何辱罵、騷擾或其他不可接受的行為，請向負責執行的社群領導者報告：
`conduct@cyber-sec.space`。
所有投訴都將受到及時且公平的審查與調查。

所有社群領導者都有義務尊重任何事件舉報者的隱私與安全。

## 執行準則

社群領導者在決定對任何違反本行為準則的行為應承擔何種後果時，將遵循以下「社群影響準則」：

### 1. 糾正

**社群影響**：使用了不恰當的語言或其他在社群中被認為是不專業或不受歡迎的行為。

**後果**：由社群領導者發出私人的書面警告，釐清違規行為的性質，並解釋該行為不適當的原因。可能會要求違規者公開道歉。

### 2. 警告

**社群影響**：單一事件或一系列行動造成的違規。

**後果**：帶有後續懲處宣告的警告。在指定的時間內，禁止與無關人員進行任何互動，包含禁止與執行行為準則的人員進行非請求的聯繫。這包括避免在社群空間以及社群媒體等外部管道進行互動。若違反這些條款，可能會導致暫時或永久性的封禁。

### 3. 暫時封禁

**社群影響**：嚴重違反社群標準，包括持續的按查不當行為。

**後果**：在指定的時間內，暫時禁止與社群進行任何形式的互動或公開交流。在此期間，不允許與涉事人員進行公開或私下的互動（包括與負責執行行為準則的人員進行非請求的互動）。若違反這些條款，將可能導致永久封禁。

### 4. 永久封禁

**社群影響**：展現出持續違反社群標準的行為模式，包含持續的按查不當行為、對特定個人的騷擾，或是針對特定群體表現出的攻擊與貶損。

**後果**：永久禁止在社群內進行任何形式的公開互動。
