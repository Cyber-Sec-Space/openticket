Hook Engine Plugins

Integrate seamlessly with your existing SOC infrastructure using zero-config extensions.

Slack Critical Notifier

Intercepts CRITICAL/HIGH Incidents and Auto-Quarantined Assets, broadcasting native payloads directly to a Slack Channel via Webhooks.

PagerDuty Escalator

Automatically triggers a PagerDuty incident when a CRITICAL severity issue is logged in OpenTicket, bridging the gap between SOC reporting and on-call response.

Jira Cloud Synchronization

Two-way structural synchronization with Jira. Generates comprehensive Jira issues from OpenTicket vulnerabilities, assigning appropriate priority codes.

Microsoft Teams Webhook

Broadcasts critical OpenTicket events as rich Adaptive Cards into designated Microsoft Teams SOC channels for immediate visibility.

Developer API

You can build your own plugins by utilizing the Hook Engine's exposed event boundaries.

export type OpenTicketPluginHooks = {
  onIncidentCreated?: (incident, config) => Promise<void>;
  onIncidentResolved?: (incident, config) => Promise<void>;
  onAssetCompromise?: (asset, config) => Promise<void>;
};
